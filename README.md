
# 使用说明

1. 将 **tesseract** 和 **BlueStacks 安卓模拟器**安装到本地

2. **第 7 行**修改为自己的 **tesseract.exe** 文件路径

3. **第 27 行**修改为屏幕截取的范围, 即 **（x, y, width, height）**

> 微信截图快捷键: Alt + a
